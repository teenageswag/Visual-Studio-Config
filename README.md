### Philosophy
Nuclear is a low-contrast theme designed for focus. It mimics the matte displays of reliable old-school terminals, avoiding neon spikes to reduce eye strain during long coding sessions.

### Palette
| Context | Color | Hex |
| :--- | :--- | :--- |
| **Logic** | Muted Gold | `#CCB333` |
| **Data** | Moss Green | `#73B82E` |
| **Content** | Off-White | `#E4E3E8` |
| **Background** | Deep Matte | `#181818` |

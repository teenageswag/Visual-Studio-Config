# Nuclear Theme ☢️

> **A high-end terminal theme from a vintage workstation.**

Nuclear is a dark, low-contrast VS Code theme designed for long coding sessions. It avoids high-saturation neon colors in favor of muted tones (gold, moss green, soft off-white) to reduce eye strain, mimicking the matte display of a reliable old-school terminal.

## Core Palette

- **Editor Background**: `#181818` (Deep matte dark)
- **UI Background**: `#212121` (Slightly lighter, strictly flat)
- **Primary Text**: `#E4E3E8` (Soft off-white)
- **Keywords**: `#CCB333` (Muted gold)
- **Strings**: `#73B82E` (Natural moss green)
- **Comments**: `#94979E` (Neutral gray)
- **Errors**: `#B82E2E` (Dried blood red)

## Design Principles

1.  **Low Contrast**: No pure blacks (`#000000`) or pure whites (`#FFFFFF`). The transition between text and background is soft.
2.  **Semantic Clarity**:
    - **Logic (Keywords)** is Gold.
    - **Data (Strings)** is Green.
    - **Content (Variables)** is White.
3.  **Minimalism**: UI elements like indent guides and line highlights are subtle (`#212121` and `#94979E33`), ensuring they guide without distracting.

## Installation

1.  Open **Extensions** sidebar in VS Code.
2.  Search for `Nuclear`.
3.  Click **Install**.
4.  Reload VS Code and select **Nuclear** from the theme list.
